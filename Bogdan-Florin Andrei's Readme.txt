1. Open Visual Studio Code
2. Import the CSV File from GitHub called FinallOneCSV.csv
3. Get the Program.py code from the file in github
4. Import it on Visual Studio Code
5. Run it
6. Click on Start Main Menu at the button
7. Click one of the first four buttons at the left (Temperature, Dew Point,Humidity and Wind Speed)
8. Choose a year on the first dropdown menu
9. Choose a value on the sencond dropdown menu
10. Press Submit Button
11. Graph showing up on the screen
12. Close the graph on the x box from top right of the chart
13. Click on Quit button on (Temperature, Dew Point,Humidity and Wind Speed) menu
Menu Closes.
